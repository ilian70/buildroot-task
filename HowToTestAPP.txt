Prerequisites

 - A working C++ development environment
 - SDL2 and SDL2_image libraries installed
 - Redis server running and accessible

A. Build the application - CMakeLists.txt is provided for building the project.

 1. Clone the repository
 2. Navigate to the project directory
 3. Create a build directory: mkdir build && cd build
 4. Run CMake: cmake ..
 5. Build the project: make


Redis server is installed and running on localhost or IP 192.168.0.100 at default port 6379.

External config file app.cfg.json is provided with default Redis server connection details.

Application can be run with optional command --config app.cfg.json to specify which config to use

B. Start the application - will display window on the

C. Open Redis CLI or provided python script ./run_redis_console.py

 if from redis-clie - Set the key as SET Image:Id 1
 if from python script - type command set_image 1 or use help to see other commands


Observe application window updating the image displayed based on the Redis key value changes.

try with SET Image:Id 2 and so on.


D. Exit the application - close the SDL window or press ESC.
