TROUBLESHOOTING TIPS:
====================

If display doesn't work:
- Check /dev/fb0 exists: ls -l /dev/fb0
- Check /dev/dri exists: ls -l /dev/dri/
- Run fbtest first (simplest test)

If OpenGL fails:
- Check glxinfo output for supported extensions
- glmark2 needs proper GPU drivers (VC4/V3D enabled in your config)
- Check /var/log/Xorg.log if X11 is running

For SDL2 testing:
- Your app should work directly since SDL2 is configured
- Make sure fbdraw.cpp uses proper SDL2 initialization
- Check if DISPLAY needs to be set for X11 mode
